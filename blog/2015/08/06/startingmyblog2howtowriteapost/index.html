Add Comments with Disqus

Create a Disqus account and configure it

xxxThe standard way of making your posts commetable in Octopress is configuring disqus for it. This is a website that will manage your comments for you, so you will be able to migrate your blogging plataform in a much easier way. The only thing you have to do is to create an account and then configure it on Octopress. In order to do this you have to set these two variables in the _config.yml file:

disqus_show_comment_count: true disqus_short_name: YOUR_DISQUS_ACCOUNT

Finally make sure that your blog entries have the comments: true option on top.
