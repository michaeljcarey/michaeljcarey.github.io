Setup Your First R Fduino

Aug 13th, 2015 1:36 pm | Comments

RFDuino is a tiny Arduino that features wireless bluetooth. The tiny shields stack and you can make any number of cool projects with this controller. I ended up buying 3 boards from them so that I can develop a servo controller. This post describes initial setup of my project. Another post will describe how I made a remotely controlled robot.

I got the standard RFduino RFD22102 with wireless bluetooth.

I also got the RFD22121 USB Shield for RFduino which is required to load code onto the RFduino.

And finally, I got the RFD22123 Servo Shield for RFduino so that I can build my future robot.

For now I will focus on the RFduino and the USB shield for prototyping. Firstly, I will follow the examples from the RFduino web site and document my progress here. This example is to load a temperature read program into the RFduino. Then I will download the RFduino Temp app onto my IPhone.
